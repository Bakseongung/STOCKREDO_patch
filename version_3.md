## v3
### 추가된 점
```yml
1 : 최근 매매 기록 추가
2 : 이동평균선(MA) 추가
3 : 보유 주식-자산 배분 원형 차트 추가
4 : 로그인, 로그아웃 추가
5 : 사용자별 랭킹 통계 제공
6 : 수수료와 슬리피지 기능 추가(+체결 미리보기)
7 : '보유 주식 탭'과  '전일 대비 탭' 세부사항 추가(평균단가, 전일 가격)
```

### 수정
```yml
1 : 매매 이유 삭제
2 : 백 테스팅 삭제
3 : '자산 리셋'을 '시스템 설정-자산 리셋'으로 경로 수정
```

### DB & Redis
```bash
# DB
updated_at = 오늘 08:40
archiver.py가 차트용 데이터를 위해 09:00~15:35 매분 스냅샷 저장
'이벤트 스케줄러'가 7일이 지난 데이터를 삭제중

# Redis
stock:{code}:price = 네이버 실시간가 ← collector_redis.py가 5초마다 갱신
stock:{code}:prev_close = 네이버 API cv field 
stock:{code}:time = 마지막 수집 시각
```

### 수수료 및 슬리피지 관련 
```bash
# 수수료 0.015%
매수: 체결금액 + 수수료 차감
매도: 체결금액 - 수수료 지급
수수료는 원 단위 올림 처리

# 슬리피지 (시뮬레이션용)
1. 국내 주식 호가 단위(틱) 기준으로 계산 (1천원 미만=1원, 5천원 미만=5원... 50만원 이상=1000원)
2. 수량 100주마다 1틱씩 추가, 최대 3틱
3. 매수는 현재가보다 위, 매도는 아래에서 체결
```
